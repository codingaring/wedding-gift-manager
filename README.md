# Wedding Gift Manager

결혼식 당일 축의금을 수납하고 관리하는 앱

## 주요 기능

### 수납 모드 (Flutter Windows 앱)

- 하객 이름, 관계, 금액을 빠르게 입력
- 3만원 ~ 30만원 빠른 입력 버튼
- 신랑 측 / 신부 측 전환 가능
- 입력 즉시 로컬 DB에 저장 (SQLite WAL 모드)
- 5분마다 자동 백업
- 삭제 시 PIN 확인으로 오조작 방지

### 관리자 모드 (웹 대시보드, 예정)

- 신랑/신부 측 필터링 조회
- 관계별, 결제수단별 분류
- 계좌이체 건 추가 입력
- 엑셀/CSV 내보내기

## 화면 구성

| 화면      | 설명                                   |
| --------- | -------------------------------------- |
| 초기 설정 | 신랑/신부 측 선택 + PIN 설정           |
| 수납 목록 | 테이블 형태 목록 + 건수 표시 + 검색    |
| 통계      | 금액 합계, 관계별 통계 (별도 탭)       |
| 추가 모달 | 이름, 금액(빠른 버튼), 관계, 메모 입력 |

## 데이터 안전

| 장치 | 방법                                   |
| ---- | -------------------------------------- |
| 1차  | SQLite WAL 모드 - 앱 충돌 시 자동 복구 |
| 2차  | 5분 자동 백업 - 기기 고장 대비         |
| 3차  | 서버 동기화 (예정) - 기기 분실 대비    |
